### :gear: Key Features

*   **:zap: Universal Account Scanner**
> Automatically extracts connected accounts from **Minecraft, Steam, Discord, and Windows Users** across **ALL drives**.
*   **:mag: Smart "Fuzzy" Target Search**
> Uses an intelligent algorithm to find username variations (e.g., `TheGamer` vs. `Th3_G4m3r_01`) and typos that standard searches miss.
*   **:floppy_disk: Advanced Forensics (USN Journal)**
> Retrieves a detailed history of **deleted and modifided files** over the last 14 days from every connected disk, exposing cleaning attempts.

### :chart_with_upwards_trend: Optimized Performance

*   **:rocket: Deep & Fast:** Scans gigabytes of data in few seconds.
*   **:broom: Smart Cleanup:** Automatically filters noise and duplicates, giving you a clean, easy-to-read report sorted by platform.

### :shield: Privacy & Security
*   **:lock: 100% Local & Safe:** The tool operates **entirely offline**. No data, logs, or results are ever sent to external servers.
*   **:briefcase: Ethical Design:** We only extract **public display names**. Sensitive data like passwords, emails, or session tokens are **never** accessed or displayed.
*   **:handshake: Peace of Mind:** Automated scanning removes the need for manual digging through personal user folders, protecting everyone's privacy.
