# :fire: RedLotus Alt Checker :fire:

### :sparkles: What's New in the Latest Update?

*   **:floppy_disk: Persistence Check Added**
> The tool now tracks **Minecraft** and **Discord** accounts even if they have been removed from the device, preserving a history of previously detected accounts.
> *Note: Steam and Hytale are currently excluded from persistence as they act primarily as an additional source of info for username matching.*
*   **:crossed_swords: Hytale Support**
> Added username detection support for **Hytale** (Launcher logs and configuration files).
*   **:bug: Stability Improvements**
> Fixed minor bugs and glitches reported on specific PC configurations for a smoother experience.

---

### :gear: Key Features

*   **:zap: Universal Account Scanner**
> Automatically extracts accounts from **Minecraft, Steam, Discord, Hytale, and Windows Users**.
*   **:mag: Smart "Fuzzy" Target Search**
> Uses an intelligent algorithm to find username variations (`TheGamer` / `Th3_G4m3r_01`) and typos that standard searches might miss.
*   **:floppy_disk: Advanced Forensics (USN Journal)**
> Retrieves a detailed history of **deleted and modified Minecraft-related files** over the last 14 days, exposing potential cleaning attempts.

### :chart_with_upwards_trend: Optimized Performance

*   **:rocket: Deep & Fast:** Scans gigabytes of data in just a few seconds.
*   **:broom: Smart Cleanup:** Automatically filters noise and duplicates, giving you a clean, easy-to-read report sorted by platform.

### :shield: Privacy & Security
*   **:lock: 100% Local & Safe:** The tool operates **entirely offline**. No data, logs, or results are ever sent to external servers.
*   **:briefcase: Ethical Design:** We only extract **public display names**. Sensitive data like passwords, emails, or session tokens are **never** accessed or displayed.
*   **:handshake: Peace of Mind:** Automated scanning removes the need for manual digging through personal user folders, protecting everyone's privacy.
